Install
------------------------------------------------------------------------------

``aws_ci_bot`` is released on PyPI, so all you need is:

.. code-block:: console

    $ pip install aws_ci_bot

To upgrade to latest version:

.. code-block:: console

    $ pip install --upgrade aws_ci_bot
